# Law Codes

Welcome to the public documentation site for the Kaharagian legal code. This collection is organized by code and article, and is published here as Markdown for easy browsing.

The navigation sidebar on the left lists every article in the collection. Use the built-in search bar to quickly jump to specific topics, or expand the sections to browse by book and chapter.

## Getting started

- Start with the fundamental principles to understand the structure of the legal order.
- Browse civil, criminal, and procedural codes by expanding their sections in the sidebar.
- Use the search field in the upper left to locate articles by keyword.

## Contributing

If you notice an issue with the text or formatting, please open a pull request on GitHub. Improvements to readability and consistency are welcome.
